Redes Sociais

 

As redes sociais são responsáveis por grande parte da integração das pessoas na Internet hoje, o potencial de disseminação de informações através destas ferramentas é inacreditável. Dessa forma o projeto Sensor4Cities integra seus recursos as principais redes sociais atualmente: Facebook e Twitter. (Durante o uso do protótipo são utilizados os perfis do laboratório GERCOM nestas redes sociais)

Facebook

 

A fanpage do projeto no Facebook é utilizada para exibit constantes atualizações das variáveis climáticas, curta a página do projeto e receba as atualizações periódicamente.

Curta nossa página no facebook agora mesmo!

Twitter

 

Siga o nosso perfil no twitter agora mesmo!

No perfil do projeto no Twitter a interação é muito maior, você pode criar uma requisação à ser disparada para os nodos sensores. Para fazer isso basta ser um seguidor do perfil do projeto e enviar um tweet comum contendo a hashtag #sensor. A resposta será enviada diretamenta através de um tweet para o seu o perfil na rede social. A figura abaixo exemplifica bem o uso dessa interação, aproveite você também!

Devido as proteções do Twitter contra ataque de usuários maliciosos, é necessário aguardar entre 1 e 2 minutos para obter a resposta. Obrigado por esperar ;-)
